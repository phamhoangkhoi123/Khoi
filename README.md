## ✨ Giới thiệu

Xin chào! Mình là **HoangKhoi**.

Đây là website cá nhân được xây dựng để chia sẻ các dịch vụ, liên kết và những dự án mình đang phát triển.

Website được thiết kế theo phong cách **Minimal • Clean • Modern** với trải nghiệm mượt mà trên cả máy tính và điện thoại.
